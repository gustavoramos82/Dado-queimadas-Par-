# Dados de Queimadas no Estado do Pará

Neste repositório vai ser feita uma análise inicial sobre detecção de focos de queimadas na vegetação, mais especificamente no Estado no Pará, no caso os dados foram obtido da [Base dos dados](https://basedosdados.org)  etambém informações sobre a mesoregião que o municipio pertence, retirando esses dados do IBGE e foi tirado algumas análises iniciais.

Nesta análise, foram considerado o periodo de 2003 até o ano de 2024, que podemos ver que nos anos de 2023/2024 teve uma aumento muito grande np número de casos, como pode ser visto no gráfico abaixo.
